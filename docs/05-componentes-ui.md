# Componentes UI

La UI usa TailwindCSS y lucide-react. Componentes base:

- Button
- Card
- Badge
- Input
- Modal
- Toast
- LoadingState
- EmptyState

Componentes funcionales:

- TicketCard
- ClaimTicketForm
- GroupPredictionBoard
- MatchScoreInput
- GroupStandingsTable
- BracketBoard
- RankingPodium
- RankingTable
- EmployeeSearch
- SellTicketPanel
- ActualResultForm

## Pendiente visual recomendado

Agregar assets propios: logo de empresa, patrón de cancha, fondos optimizados y un set de banderas SVG si se decide no depender de emojis.
